# Astra Flight Logistics

A real-time flight passenger monitoring dashboard built with Vue 3, TypeScript, and Naive UI.
Displays live PAX figures per flight with capacity alerts, sortable data tables, and multi-criteria search.

## Features
- Live passenger counts (male / female / child / infant) per flight
- Automatic capacity alerts — near-capacity (yellow) and overloaded (red) row highlighting
- Search by date or by carrier + flight number
- Responsive, clean UI inspired by aviation operations tools

## Tech Stack
- Vue 3 + TypeScript
- Naive UI component library
- Vite
- REST API integration with JWT-style key auth
